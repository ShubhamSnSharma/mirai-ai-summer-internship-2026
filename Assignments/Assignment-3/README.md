# 🌌 Assignment 3 - The Memory Vault (Stateful Chatbot)

This project is an upgraded version of Assignment 2 completed during the MirAI School of Technology Virtual Summer Internship 2026.

The focus of this assignment was to convert a **stateless chatbot** into a **stateful chatbot** using `st.session_state`, allowing the conversation history to persist across Streamlit reruns.

---

## Features

- Stateful conversation using `st.session_state`
- Chat interface using `st.chat_input()` and `st.chat_message()`
- Multiple AI personalities
- Sidebar settings
- Clear conversation option
- Google Gemini API integration

---

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- python-dotenv
